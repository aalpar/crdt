# GC Policy Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add `DeltaStore.Dots()` and a `replication.GC()` free function that removes deltas all peers have acknowledged.

**Architecture:** `Dots()` exposes the store's keys as a snapshot. `GC()` iterates that snapshot, checks `PeerTracker.CanGC()` per dot, and removes passing entries. Triggered by the caller after every `Ack`.

**Tech Stack:** Go, stdlib only.

**Design doc:** `docs/plans/GC-POLICY-DESIGN.md`

// Dots returns a snapshot of all stored dots.
func (s *DeltaStore[T]) Dots() []Dot {
	dots := make([]Dot, 0, len(s.deltas))
	for d := range s.deltas {
		dots = append(dots, d)
	}
	return dots
}
```

package replication

import "github.com/aalpar/crdt/dotcontext"

// GC removes deltas from the store that all tracked peers have
// acknowledged. Returns the number of deltas removed.
func GC[T any](store *dotcontext.DeltaStore[T], tracker *PeerTracker) int {
	var removed int
	for _, d := range store.Dots() {
		if tracker.CanGC(d) {
			store.Remove(d)
			removed++
		}
	}
	return removed
}
```
